# Trident300
Voron Trident 300 configuration backup.
Build from a Formbot kit. Has Chaotic labs v1 tap. Canbus board - SB2040, Octopus v1.1 STMF446, Raspi Pi 4B+. Running klipper through the great klippain variant.
